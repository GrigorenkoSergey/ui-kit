Стилизация

Точно такой же, но с квадратными ячейками

У каждого компонента есть свои исходные стили, написанные разработчиком. Они хранятся в статическом свойстве CustomCalendar.defaultSheets Это массив, который можно дополнить дополнительной таблицей стилей или полностью переписать для всех компонент глобально. Это нужно сделать в самом начале при первом подключении компонента. Ничего не мешает добавить дополнительно классы для каждого компонента. К примеру, вы решили, что некоторые календарики должны быть с квадратными ячейками... Для этого можно добавить класс для хоста.

<custom-calendar data-testid="styles-customized-rect" class="rect-cells"></custom-calendar> <script defer> // Где-то в JS-файлах... const additionalSheet = new CSSStyleSheet(); // по мне проще всего создать в отдельном файле и импортировать ее как строку, additionalSheet.replaceSync(` :host(.rect-cells) { .date-cell, .month-cell, .year-cell, button { border-radius: 4px; } }`); CustomCalendar.defaultSheets = [CustomCalendar.defaultSheets[0], additionalSheet]; </script>

Полностью перезаписанные стили

Можно перезаписать стили отдельно для уже существующего компонента. Для этого у каждого компонента будет использоваться теневое дерево и adoptedStyleSheets.

<custom-calendar data-testid="styles-customized-full"></custom-calendar> <script> const fullyCustomized = document.querySelector("[data-testid='styles-customized-full']"); const fullyCustomizedSheet = new CSSStyleSheet(); // fullyCustomizedCSS - импортированный как строка файл CSS fullyCustomizedSheet.replaceSync(fullyCustomizedCSS); fullyCustomized.shadowRoot.adoptedStyleSheets = [fullyCustomizedSheet]; </script>
